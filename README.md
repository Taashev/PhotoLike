# Mesto
-----------------------------------------------

### Обзор
Mesto - веб-приложение где вы можете хранить свои фото и получать лайки от друзей.
-----------------------------------------------
### Технологии при разработке:

* БЭМ
* Файловая структура Nested
* Flex-box
* Grid layout
* Позиционирование
* @font-face 'Inter' (400, 500, 900)
* @media
* @import
* Отзывчивая верстка 320px - 1280px+
* JavaScript
* ООП
* Webpack
-----------------------------------------------
